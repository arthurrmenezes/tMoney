# tMoney

Este repositório contém a API REST do tMoney, responsável pela gestão financeira dos usuários. A API permite o registro de receitas e despesas, a criação e vinculação de transações a contas de crédito e débito, a categorização de transações e a geração de resumos financeiros, entre outras funcionalidades.  

O back-end foi desenvolvido utilizando C#, .NET 8, PostgreSQL e Docker.
